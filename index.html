pipeline

From bits to
base pairs

BioVault follows the same pipeline used by real DNA storage research — from Microsoft and UW to JCVI.

01
compress
Pre-compress your file with zstd or gzip. Repetitive data like text, JSON, or HTML shrinks up to 98% — directly reducing synthesis cost.
--compress
02
encode
Bytes are converted to 2-bit base pairs (00→A, 01→T, 10→C, 11→G), chunked into 150-base oligos with index headers and MD5 checksums.
encoder.py
03
error correction
Reed-Solomon ECC is applied per fragment — the same algorithm used in CDs and QR codes. Corrects up to 5 byte errors per 150-base oligo.
--ecc
04
export FASTA
Fragments are exported as standard FASTA format — directly uploadable to Twist Bioscience or IDT for physical DNA synthesis.
output.fasta
05
primer design
PCR primer pairs are generated for each fragment. In wet lab, primers allow selective retrieval of only the data you need — a biological search query.
--primers
06
decode
Sequencing reads (e.g. from Oxford Nanopore MinION) go back through the pipeline. ECC corrects errors, checksums verify fragments, file is recovered exactly.
biovault decode

features

Everything in one library

A complete DNA storage pipeline you can run from the command line or import as a Python library.

Python API + CLI
Use as a library or run from the terminal. Encode, decode, simulate, stats — all in one command.
Reed-Solomon ECC
Corrects up to 5 byte errors per fragment — handles the ~1% error rate typical of real nanopore sequencing.
zstd compression
Auto-selects best codec (zstd, gzip, or raw). Reduces synthesis cost by up to 98% on compressible data.
GC content analysis
Validates synthesis-readiness of sequences. Flags fragments with GC content outside the 40–60% sweet spot.
PCR primer design
Designs forward/reverse primer pairs per fragment. Validates Tm, GC%, homopolymer runs. Exports synthesis-ready CSV.
FASTA export
Standard bioinformatics format with index headers and checksums. Upload directly to Twist Bioscience or IDT.

quickstart

Up in 30 seconds

Clone the repo, install dependencies, encode your first file.

# clone and install
git clone https://github.com/Altf4ah/biovault
cd biovault
pip install -e ".[dev]"

# encode a file to DNA
python -m biovault encode myfile.pdf --compress --ecc --primers

# decode it back
python -m biovault decode myfile.pdf.fasta --out recovered.pdf

# simulate 1% sequencing error rate
python -m biovault simulate myfile.pdf.fasta --error-rate 0.01

# run all 20 tests
pytest tests/ -v
